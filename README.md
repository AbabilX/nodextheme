# Murad AMOLED

A minimal **AMOLED black** Visual Studio Code theme inspired by GitHub Dark, with vivid syntax colors on a pure black canvas.

## Preview

> True black (`#000000`) UI · GitHub-inspired syntax · AMOLED-friendly contrast

```text
Editor background: #000000
Accent / focus:    #7ee787
Keywords:          #ff7b72
Functions:         #b392f0
Strings:           #a5d6ff
```

## Features

- Pure AMOLED black editor and chrome (`#000000`)
- Vivid syntax highlighting inspired by GitHub Dark
- Semantic highlighting enabled
- Clear contrast for long coding sessions on OLED displays
- Consistent UI tokens for sidebar, status bar, terminal, and debug views

## Installation

### From VSIX

1. Download the `.vsix` package.
2. In VS Code / Cursor: **Extensions** → `⋯` → **Install from VSIX…**
3. Select `murad-amoled-1.0.0.vsix`.
4. Open **Preferences → Color Theme** and choose **Murad AMOLED**.

### From Extension Development Host

1. Open the `murad-amoled` folder in VS Code.
2. Press **F5** to launch the Extension Development Host.
3. Select **Murad AMOLED** from the Color Theme picker.

### Command line

```bash
code --install-extension murad-amoled-1.0.0.vsix
```
